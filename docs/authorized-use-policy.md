# Authorized Use Policy

Exodia must be used only in environments that are owned, controlled, or explicitly authorized for assessment.

Before each scan, the user must confirm:

```text
Confirmo que tenho autorizacao para analisar este alvo.
```

The target record must document the authorized scope. Scope should include:

- Asset owner or approving party.
- Environment name.
- Allowed target URLs, hosts, or IPs.
- Time window or recurring authorization.
- Modules or categories allowed.
- Any operational limits.

If authorization is unclear, do not scan.
