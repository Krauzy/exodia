# Exodia Frontend

Electron, React, TypeScript, Vite, TailwindCSS, React Query, Zustand, TanStack Table, Recharts, Monaco Editor, and Lucide React.

## Development

```bash
npm install
npm run dev
```

Log in or register from the first screen. The frontend stores the bearer token in local storage and sends it to the
backend for targets, scans, reports, SRE checks, and Pentest checks.

Run the desktop shell:

```bash
npm run electron:dev
```

The renderer reads `VITE_API_BASE_URL`; copy `.env.example` to `.env` when the API is hosted outside the default
`http://127.0.0.1:8765`.

The Electron main process starts the Python backend at `127.0.0.1:8765` only when no external backend URL is
configured. To use an already running backend, set `EXODIA_BACKEND_URL` and run:

```bash
npm run electron:external
```

## Installers

Installers are generated by the repository GitHub Actions workflow on Windows, Linux, and macOS. The pipeline builds a platform-native backend executable with PyInstaller, places it in `frontend/resources/backend`, and then runs `electron-builder`.

Local packaging still works as a development fallback. If `frontend/resources/backend/exodia-backend` is absent, the packaged app falls back to a local Python runtime and `backend-source`.
